# CampaignOS

**Free. Open Source. Built for creators and businesses.**

A professional campaign management system powered by local AI (Ollama).

## What is CampaignOS?

A clean, powerful tool to run high-quality marketing and content campaigns without paying for SaaS tools.

- Research → Content → Distribution planning
- Works via Telegram bot + beautiful web dashboard
- 100% local and private
- Completely free forever

## Quick Start

```bash
make setup
make model          # Downloads llama3.1:8b
make all            # Bot + dashboard + scheduler at http://localhost:8000
```

## Features

- Structured Campaign Protocol (Research, Content, Schedule)
- Web Dashboard to manage campaigns
- Telegram bot interface
- Full persistence with SQLite
- Works with any local Ollama model

## Philosophy

We believe powerful marketing tools should be **free and accessible** to everyone — not locked behind expensive subscriptions.

This project is shipped with that intention.
